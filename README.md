# RockMech_SlopeCal
Slope Calculator for Rock Mech

## 使用说明

### 计算文件准备

Step.1 文件路径不能出现中文  
Step.2 将文件存成CSV格式  
Step.3 CSV文件列数为2~4，第一行为列名，应保证不存在缺项行或NULL值。示例如下：
